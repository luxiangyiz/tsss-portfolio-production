# ZWD Portfolio Theme

`ZWD Portfolio` 是钟伟达个人简历网站使用的独立 WordPress 区块主题。

主题的初始结构与区块主题实践参考了 Ollie 1.6.0，并针对简体中文、黑白编辑风、项目卡片和求职场景重新设计。主题不包含 Ollie 的商业资源、远程字体、示例图片或 WooCommerce 模板。
