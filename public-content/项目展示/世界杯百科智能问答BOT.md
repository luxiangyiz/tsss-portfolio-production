# 世界杯百科智能问答 BOT

## 项目背景

世界杯相关信息覆盖赛程、球队、球员、历届赛果、赛事规则和比赛场馆等多个主题，资料来源分散、时间跨度较长。普通搜索需要在不同页面之间反复查找，而通用大模型也可能出现资料滞后、事实混淆或脱离来源作答的问题。

项目使用 Dify 搭建世界杯百科智能问答 BOT，并配置专属 RAG 知识库，把分散资料整理为可检索内容，再通过检索增强生成让模型优先依据知识库回答。

## 负责内容

1. 整理世界杯主题资料，处理不利于检索的排版和内容结构，并完成知识库导入前的文档预处理。
2. 在 Dify 中配置知识库切片与检索策略，观察不同设置对召回内容和回答完整度的影响。
3. 设计和迭代问答 Prompt，明确主题范围、资料优先级和回答方式。
4. 调试赛程、球队、球员、历届赛果、赛事规则和球场信息等问答场景，并检查连续追问表现。
5. 完成 Dify 应用编排，生成并验证网页或公众号嵌入方式。

## 技术栈

- Dify：知识库、检索、大模型对话和应用入口编排。
- RAG：在生成回答前检索世界杯专属资料，降低回答脱离资料的风险。
- Embedding 向量检索：根据问题语义召回相关知识分块。
- Prompt Engineering：约束回答范围并引导模型结合检索结果作答。
- 知识库分片与检索策略：平衡主题完整性、检索上下文和无关内容。
- 网页与公众号嵌入：验证应用面向用户的轻量化接入方式。

## 当前成果

- 跑通从资料整理、知识库配置、问题检索到模型回答的基本链路。
- 覆盖赛程、球队、球员、历届赛果、赛事规则和球场信息等多类问题。
- 完成多轮问答场景调试和网页或公众号嵌入方式验证。
- 沉淀资料质量、知识库切片、检索结果与 Prompt 之间关系的实践经验。
